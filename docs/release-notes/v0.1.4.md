# AzImg CLI v0.1.4

Patch release for Windows packaged authentication.

## Changes

- Fixed Windows packaged `azimg doctor --verify-auth` failures where Azure CLI
  token acquisition could report `Failed to load python executable` even though
  `az account get-access-token` worked in the shell.
- Added release validation coverage for the packaged Windows executable so the
  installed artifact is tested instead of relying on `dotnet run`.
- Added the `uninstall` command for removing the installed `azimg` executable.
