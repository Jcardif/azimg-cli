# AzImg CLI v0.2.1

Patch release for long prompt ergonomics.

## Changes

- Added `azimg generate --prompt-file <path>` and
  `azimg edit <input-file-or-folder> --prompt-file <path>` to read long prompts
  from text files while preserving line breaks.
- Added the bundled AzImg agent skill version to `azimg version` output.
- Updated CLI help, documentation, and the AzImg agent skill for prompt files.
